Advanced MAC Authentication Help

MAC Authentication

This page lets you block the network access privilege of the specified stations through the WNDAP360 Wireless Access Point.

Select the Turn Access Control On check box if you want the Access Point to interact only with stations present in the Trusted Wireless Stations table. This provides an additional layer of security.

Import MAC Address List from a file

This section allows you to get the Wireless Stations list from a file in your local disk. You can select one of the following options:

    • Replace: The Access Point will replace the current list of wireless Stations with a new list from the selected file.
    • Merge: The Access Point will merge the current list of wireless Stations and the new list from the selected file to create a combined list.

Click Browse To locate and select the previously saved file containg the list of Wireless Stations.This file shall be in text format with one MAC address on each line.

Click APPLY To save the change.

Select a database from the Select Access Control Database list. You can select one of the following options:

    • Local MAC Address Database: The Access Point will use the local MAC address table for Access Control.
    • Remote MAC Address Database: The Access Point will use the MAC address table located on an external Radius server on the LAN for Access Control..

Trusted Wireless Stations

This table lists the stations that have been allowed access to the network through this Access Point.

Click Delete to delete a wireless station from the Trusted Wireless Stations list.

Enter a MAC address and Click Add to add the MAC address of the wireless stations to the Trusted Wireless Stations list.

Available Wireless Stations

This table lists the Wireless stations that are present in the vicinity of the Access Point. Click Refresh to update this list in the table.

Select a station in the Available Wireless Stations table and click Move to move the station to the Trusted Wireless Stations table.

  • Use the Save as button to save the Trusted Wireless Stations list to the file on the disk.
